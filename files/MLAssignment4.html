Now lets train a SVM model. This is done using scikit-learn, the default kernel is a Radial Basis Function (RBF). am using a RBF kernel because the data is not linearly separable. Below the kernel is defined.

$\phi(r) = e^{-(\gamma r)^2}$,

where $r = ||x-x'||$

$\gamma$ represents the influence a single training point will have. A large gamma means the Gaussian kernel has a large variance, and likewise, a large gamma implies a small variance.

Next, the kernel trick will be used to transform the data into a 3D space. The kernel function I am using to do this is:

$z = \sqrt{x^2 + y^2}$

Where x and y are a single point from input 1 and input 2.

Below is the transformation of the data.

# Training the new 3D model
model = svm.SVC(kernel='linear')
clf = model.fit(train_x, train_y)


# The equation of the separating plane is given by all x so that np.dot(svc.coef_[0], x) + b = 0.
# Solve for w3 (z)
z = lambda x,y: (-clf.intercept_[0]-clf.coef_[0][0]*x -clf.coef_[0][1]*y) / clf.coef_[0][2]

tmp = np.linspace(-2,2,30)
x,y = np.meshgrid(tmp,tmp)

fig = plt.figure()
ax  = fig.add_subplot(111, projection='3d')
