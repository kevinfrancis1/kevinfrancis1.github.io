Statistical Modeling

This was a course I took that covered hypothesis testing of means, variances, proportions, and making inference from sample data. Below are three projects from this course. All were done in R and are presented using R markdown.

  • Project 1 - This project looks at making inference on hospital admission price based on the region it is located in.
  • Project 2 - This project also makes inference on a hospitals addmission price, but looks to see how it depends on the category of admission and the severity of illness. This is done using one and two-way anova.
  • Project 3 - This project attempts to predict housing prices based on multiple predictors. This is done using multiple linear regression.
